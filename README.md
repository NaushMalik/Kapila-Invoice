# Kapila Restaurant Billing Management System 📊🌯

**College Project** - Full-stack POS/Invoice system

## 🚀 Quick Start

### 1. Prerequisites
- Python 3.12+
- MySQL (XAMPP recommended)
- pip install -r requirements.txt (done)

### 2. MySQL Setup
```
1. Update setup_mysql.bat (MYSQL_ROOT_PASSWORD = your root pw)
2. Run: setup_mysql.bat
3. Copy .env.example → .env (update DB creds)
```

### 3. Run App
```
python app.py
```
- Open: http://localhost:5001
- **Login**: `admin` / `admin123`

### 4. Test Features
1. **Dashboard** → Sales overview
2. **New Invoice** → Add customer/items → Save → PDF/WhatsApp/Email
3. **Products** → CRUD test
4. **Settings** → Update business/SMTP
5. **Invoices** → List/search/export
